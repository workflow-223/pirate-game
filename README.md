# Pirate's Bounty

HTML5 Canvas arcade game where players pilot a pirate ship through treacherous seas, battling enemy AI ships and collecting treasure across waves.

## Features

- Top-down pirate ship controller with keyboard/mouse input
- AI enemy ships with different behavior patterns
- Treasure collection and wave-based progression
- Score tracking and game state management

## Deployment

Deployed to GitHub Pages at:
`https://workflow-223.github.io/pirate-game/`

## Local Development

Simply open `index.html` in a browser - no build step required.
